Rules for the Host

Read these aloud before kicking off the game so everyone is on the same page.

1

Five puzzles each

Every participant is assigned five puzzles. The host calls names in order and reveals the puzzle on screen.

2

30-second timer

The countdown auto-starts the moment a puzzle appears. The ring turns yellow at 10s and red at 5s — answer before it hits zero. Use ⏸ to pause if needed.

3

Simple scoring

+5 fully correct · +3 correct after hint · +2 partially correct · −1 wrong guess.

4

Hints cost points

If a player is stuck, the host can reveal the hint — but the max score drops from +5 to +3. Use hints to keep the game moving.

5

No Googling

No browser tabs, AI assistants, or whispering to colleagues off-camera. Cameras stay on if possible.

6

Track on the scoreboard

Use the +5 / +3 / +2 / −1 buttons on the Play tab after each answer. The leader is highlighted in gold automatically.

7

Settle ties fairly

If two or more players tie, switch to the Tie-Breaker tab. First correct answer wins the round.

8

Keep it kind

No mocking wrong answers. The whole point is laughing together, not at each other. Celebrate clever guesses, even when they're wrong.
